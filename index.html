JavaScript ඉතිහාසය & හැදෑරීම

JavaScript කියන්නේ web page වල අන්තර්ක්‍රියාව සහ වැඩි දියුණු විශේෂාංග ලබා දෙන programming language එකක්. 1995 වර්ෂයේ Brendan Eich විසින් නිර්මාණය කරන ලදී.

  • Web browsers වල run වෙන first programming language එක
  • Dynamic content generate කරන්න ඉතා සුදුසුයි
  • DOM manipulation, events handling සහ asynchronous programming සමඟ
  • JavaScript (JS) කියන්නේ programming language එකක්.
  • මුලින්ම 1995දී Brendan Eich (Netscape) විසින් 10 දිනක් ඇතුළත නිර්මාණය කළා.
  • මුල් නම Mocha → LiveScript → JavaScript ලෙස වෙනස් වුණා.
  • Java language එකට සම්බන්ධයක් නෑ. නම marketing එකක් විතරයි.
  • JS භාවිතා කරන්නේ HTML + CSS එක්ක, websites interactive කරන්න.

HTML ගැන ඉගෙනිම

HTML කියන්නේ web page එකේ structure සහ semantic markup එක සදහා භාවිතා කරන markup language එකකි.

  • Headings, paragraphs, lists, links, images
  • Forms, tables, semantic tags
  • Accessibility and SEO සඳහා වැදගත්
  • HTML (HyperText Markup Language) යනු වෙබ් පිටු නිර්මාණය කිරීම සඳහා භාවිතා වන ප්‍රමුඛ භාෂාවකි. එය වෙබ් පිටුවක ආකෘතිය (structure) සහ අන්තර්ගතය (content) නිර්වචනය කරයි. HTML යනු වෙබ් අඩවි සඳහා පදනම වන අතර, CSS (Cascading Style Sheets) සහ JavaScript වැනි තාක්ෂණයන් සමඟ එක්ව භාවිතා වේ. මෙම පාඩමෙන් HTML ගැන සම්පූර්ණ විස්තරයක් ලබාදෙන අතර, එය ආරම්භකයින්ට පහසුවෙන් තේරුම් ගත හැකි ආකාරයෙන් ඉදිරිපත් කරමි.
  • HTML යනු මොකක්ද?
  • HyperText: වෙබ් පිටුවල එකිනෙකට සම්බන්ධ වන ලින්ක් (links) හරහා ගමන් කිරීමට ඉඩ සලසයි.
  • Markup Language: HTML භාවිතා කරන්නේ ටැග් (tags) භාවිතයෙන් අන්තර්ගතයේ ආකෘතිය විස්තර කිරීමටයි.
  • HTML යනු වෙබ් පිටුවක "අස්ථි ආකෘතිය" (skeleton) ලබා දෙන භාෂාවකි, එහිදී CSS මගින් එය අලංකාර කරන අතර JavaScript මගින් එය ගතික (dynamic) කරයි.
  • HTML හි මූලික ආකෘතිය
  • HTML ගොනුවක මූලික ආකෘතිය පහත ආකාරයෙන් පවතී:
  • <!DOCTYPE html>
  • <html lang="en">
  • <head>
  • <meta charset="UTF-8">
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • <title>මගේ පළමු වෙබ් පිටුව</title>
  • </head>
  • <body>
  • <h1>සාදරයෙන් පිළිගනිමු!</h1>
  • <p>මෙය මගේ පළමු HTML පිටුවයි.</p>
  • </body>
  • </html>
  • විස්තරය:
  • <!DOCTYPE html>: HTML ගොනුව HTML5 ආකෘතියෙන් ලියා ඇති බව බ්‍රව්සරයට දන්වයි.
  • <html lang="en">: HTML ගොනුවේ මූලික ටැගය වන අතර, භාෂාව (language) නිර්වචනය කරයි (මෙහිදී ඉංග්‍රීසි).
  • <head>: වෙබ් පිටුවේ මෙටා තොරතුරු (metadata) අඩංගු වේ. උදා: <meta charset="UTF-8"> යනු භාවිතා කරන අක්ෂර කට්ටලයයි.
  • <title>: බ්‍රව්සරයේ ටැබ් එකේ හෝ මාතෘකාවේ දිස්වන නම.
  • <body>: වෙබ් පිටුවේ දිස්වන අන්තර්ගතය (content) මෙහි ඇතුළත් වේ.
  • HTML ටැග් (Tags)
  • HTML ටැග් යනු HTML හි මූලික ගඩොල් (building blocks) වේ. ටැග් යනු < සහ > අතර ලියැවෙන වචන හෝ අකුරු වන අතර, ඒවා යුගල වශයෙන් (opening and closing tags) භාවිතා වේ.
  • උදාහරණ:
  • <p>: ඡේදයක් (paragraph) නිර්මාණය කරයි.
  • <h1>: ප්‍රධාන ශීර්ෂය (heading) නිර්මාණය කරයි.
  • <div>: අන්තර්ගතය බෙදීමට භාවිතා වන කන්ටේනරයකි.
  • <img>: රූපයක් එක් කිරීමට භාවිතා වේ (මෙය self-closing ටැගයකි).
  • ටැග් ආකෘතිය:
  • <opening_tag>අන්තර්ගතය</closing_tag>
  • උදා: <p>මෙය ඡේදයකි.</p>
  • Self-Closing Tags:
  • සමහර ටැග්වලට closing tag අවශ්‍ය නොවේ:
  • <img src="image.jpg" alt="විස්තරය">
  • <br>
  • <hr>
  • HTML ගුණාංග (Attributes)
  • ටැග්වලට ගුණාංග එක් කළ හැකි අතර, ඒවා ටැග්වල තවත් තොරතුරු ලබා දෙයි.
  • උදාහරණ:
  • <a href="https://www.example.com">මෙතන ක්ලික් කරන්න</a>
  • href යනු ගුණාංගයකි (attribute), එය ලින්ක් එකේ ගමනාන්තය දක්වයි.
  • පොදු ගුණාංග:
  • id: එක් එක් ටැගයට ඒකීය හඳුනාගැනීමක් ලබා දෙයි.
  • class: CSS හෝ JavaScript හරහා ටැග් එකකට style හෝ ගතික බව එක් කිරීමට.
  • src: රූප හෝ ගොනු සඳහා ගමනාන්තය.
  • alt: රූපයක විස්තරය (accessibility සඳහා).
  • HTML හි ප්‍රධාන ටැග්
  • ශීර්ෂ (Headings):
  • <h1>ප්‍රධාන ශීර්ෂය</h1>
  • <h2>උප ශීර්ෂය</h2>
  • <!-- h1 සිට h6 දක්වා ඇත -->
  • ඡේද (Paragraphs):
  • <p>මෙය ඡේදයකි.</p>
  • ලින්ක් (Links):
  • <a href="https://www.example.com">වෙබ් අඩවියට පිවිසෙන්න</a>
  • රූප (Images):

CSS ගැන ඉගෙනිම

CSS (Cascading Style Sheets) යනු වෙබ් පිටුවල අන්තර්ගතයට ආකෘතිකරණය (styling) සහ රූපලාවන්‍ය (visual appeal) එක් කිරීමට භාවිතා වන භාෂාවකි. HTML ලබා දෙන ව්‍යුහයට (structure) CSS මගින් වර්ණ, පිරිසැලසුම (layout), සහ ගතික බව එක් කරයි. CSS භාවිතයෙන් වෙබ් අඩවි responsive (උපාංග විවිධත්වයට ගැලපෙන) සහ ආකර්ශනීය බවට පත් කළ හැක.

  • වර්ණ, ෆොන්ට, පරතර, පසුබිම් (backgrounds) එකතු කිරීම
  • Box model, flexbox, grid layouts
  • Media queries භාවිතයෙන් responsive design
  • CSS යනු මොකක්ද?
  • CSS යනු Cascading Style Sheets යන්නෙහි කෙටි යෙදුමකි.
  • "Cascading" යනු styles එකිනෙකට බලපෑම් කරන ආකාරයයි (උදා: එක් style එකක් තවත් එකකට override විය හැක).
  • CSS HTML හි අන්තර්ගතයට රූපලාවන්‍ය එක් කරන අතර, responsive design සඳහා බහුලව භාවිතා වේ.

  • CSS ඉතිහාසය

  • 1996: CSS Level 1 (CSS1) නිකුත් වුණා (W3C).
  • 1998: CSS Level 2 (CSS2) – positioning, z-index එකතු වුණා.
  • 2011: CSS3 – animations, flexbox, grid, media queries එකතු වුණා.
  • වර්තමානය: CSS4 නැත, නමුත් CSS modules (flexbox, grid, custom properties) ලෙස නව features එකතු වෙනවා.

  • Cascading: Styles එකිනෙකට බලපායි (specificity, inheritance).
